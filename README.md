# Inventory System

## Project Title
Inventory System

## System Description
A database system to manage products, suppliers, categories, and orders for an inventory.

## Tables Description
- **Suppliers** – stores supplier information  
- **Categories** – product categories  
- **Products** – product details and stock  
- **Orders** – order records  
- **OrderDetails** – products included in orders

## Features Implemented
- Database creation and design  
- Normalization (up to 3NF)  
- Insert sample data (15 records per table)  
- SQL queries: SELECT, INSERT, UPDATE, DELETE, JOIN, SUBQUERY  
- Indexing for faster search

## Reflection
I learned how to design a relational database, normalize tables, write SQL queries, insert sample data, and use indexing to improve performance.
